How to add items

Edit MEDIA in app.js.

  • Internet Archive item: use { ia: "plan-c-2012", title: "Plan C (2012)" }
  • IA details link also works: { ia: "https://archive.org/details/plan-c-2012" }
  • Direct file: { url: "https://example.com/video.mp4", title: "My Video" }

The script uses the IA metadata API to find real downloadable files. It prefers MP4/WebM for playback and still provides a direct download for MKV/AVI.
